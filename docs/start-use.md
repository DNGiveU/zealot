## 支持场景

适用于Java web项目，`JDK1.6`及以上

## 安装集成

以下仅列举`Maven`和`Gradle`的使用方式，你也可以直接将此jar包加入到你的`classpath`中。

### Maven

```xml
<dependency>
    <groupId>com.blinkfox</groupId>
    <artifactId>zealot</artifactId>
    <version>1.1.1</version>
</dependency>
```

### Gradle

```
compile 'com.blinkfox:zealot:1.1.0'
```
